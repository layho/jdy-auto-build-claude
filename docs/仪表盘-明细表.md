# 仪表盘 - 明细表

> 来源: https://hc.jiandaoyun.com/doc/10969

## 功能简介

明细表以表格形式呈现和管理数据，是仪表盘中的一种图表组件。

## 设置步骤

### 1. 新建仪表盘 → 明细表
创建仪表盘，选择「明细表」图表类型，指定数据源。

### 2. 设置数据获取权限

| 权限类型 | 说明 |
|---------|------|
| 表单中的全部数据 | 成员可查看全部数据和全部显示字段 |
| 继承成员对表单的权限 | 继承字段权限（列）和数据权限（行） |

### 3. 添加计算字段
对数字字段进行简单计算（+、-、*、/），不支持函数。

### 4. 添加显示字段
支持：全选批量添加、逐个勾选、拖拽排序。可添加表单字段、系统字段、计算字段。

### 5. 过滤条件
设置过滤条件，仅展示部分需要的数据。

## 功能配置

| 配置项 | 说明 |
|-------|------|
| 操作权限 | 可导出权限、继承成员对单条数据的操作权限（编辑/删除/复制等） |
| 数据显示 | 显示序号、显示前 N 条数据 |
| 列冻结 | 固定列维度，滑动时始终可见 |
| 条件格式 | 区间色、渐变色、数据条 |
| 列宽调整 | 拖拽列分界线调整列宽 |

## 组件样式

支持设置：整体背景色、标题颜色、表格主题色、表头/内容文字颜色、内容对齐方式

## 明细表 vs 数据管理表

- **明细表**：主要用于展示数据
- **数据管理表**：侧重重度数据管理操作
