# 仪表盘 - 第一张仪表盘

> 来源: https://hc.jiandaoyun.com/doc/13811

## 功能简介

仪表盘用于对收集到的数据进行分析，并支持将分析结果进行多维度、实时地展示。

## 设计步骤（4步）

1. **新建仪表盘**
2. **添加图表和数据源**
3. **添加维度和指标**
4. **效果查看**

## 详细操作

### 2.1 新建仪表盘
1. 进入需要搭建仪表盘的应用
2. 选择**新建仪表盘**，修改名称

### 2.2 添加图表和数据源
1. 在仪表盘左侧编辑界面选择**统计表**
2. 添加数据源为表单中的数据表
3. 修改统计表名称

### 2.3 添加维度和指标
1. 选择图表类型（饼图、柱形图、折线图等）
2. 添加字段至**维度**和**指标**
3. 修改指标显示名
4. 保存

## 关键概念

| 概念 | 说明 |
|------|------|
| 仪表盘 | 数据分析和多维度实时展示的工具 |
| 统计表 | 用于数据统计分析的图表类型（含饼图、柱形图、折线图等） |
| 维度 | 数据分类的维度字段（如按性别分组） |
| 指标 | 用于计量的数值字段（如计数、求和、比例） |
| 数据源 | 仪表盘所引用的表单数据来源 |

## 重要提示
- 仪表盘需在应用内新建，数据源引用自已有表单
- 支持多图表组合展示，包括统计表、明细表等
- 支持饼图、柱形图、折线图、面积图、透视表等多种图表类型
