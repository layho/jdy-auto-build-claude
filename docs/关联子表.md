# 关联子表

> 来源: https://hc.jiandaoyun.com/doc/21272

## 功能简介

关联子表字段可绑定一张关联表，以表格形式呈现于主表内，方便查看、编辑、导出、打印关联表数据，并支持同时填报主表和关联表数据。

## 应用场景

- 客户管理中查看客户详情时，同时查看跟进记录、联系人、订单数据
- 订单管理中录入订单明细，便于出库/退货时拆分处理
- 采购管理中筛选供应商明细进行合并付款

## 设置步骤

### 绑定关联表
两种方式：
1. **绑定已有表单** — 从下拉列表选择已通过关联数据字段建立关联的表单
2. **从空白新建** — 输入表单名称，点击「设计关联表」新建绑定

注意：一张表单最多被 100 张表单关联

### 显示字段
- 最多 50 个显示字段
- 不支持：分割线、查询、选择数据、文字识别、按钮、子表单、关联子表

### 数据操作

**查看/编辑主数据时**：
| 操作 | 说明 |
|------|------|
| 可新增 | 新增关联表数据 |
| 可导入 | 向关联表导入数据 |
| 可导出 | 导出关联表数据 |
| 可直接编辑已有数据 | 在表格中直接修改 |
| 可删除已有数据 | 在表格中直接删除 |
| 可查看数据详情 | 按成员关联表权限展示 |

**新增主数据时（随主数据一同新增）**：
- 提交时每条数据自动建立关联并入库
- 一张表单仅支持 1 个关联子表启用此功能
- 最多支持一同新增 **500 条**数据

### 快速填报
- 新增记录 或 编辑已有记录两种模式
- 支持复制粘贴Excel数据、直接编辑单元格、批量编辑、下拉填充
- 数据达到 500 行上限时仅支持编辑已有记录

## 注意事项

1. 删除关联子表字段或删除主数据，均不会级联删除关联表数据
2. 数据独立存储，数据量上限由关联表决定
3. 若需级联删除或同步更新，可借助智能助手Pro实现
4. 关联子表字段不支持作为子字段，不可拖入子表单
5. 外链不可见
