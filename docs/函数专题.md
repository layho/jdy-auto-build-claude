# 函数专题

> 来源: https://hc.jiandaoyun.com/doc/13956

## 常用函数类型

- **MAPX**：跨表单/跨条件数据查询与引用，多条件匹配的核心函数
- **DATEDIF**：日期差计算，支持年/月/日三种粒度
- **IF嵌套**：条件分支逻辑，多层判断时的核心工具
- **INT/VALUE**：数值处理，金额拆分和类型转换
- **MID**：文本截取，身份证号信息提取

## 函数应用

| 函数 | 应用场景 |
|------|---------|
| MAPX | 多条件数据查询、查找上次录入值、显示最新数据 |
| DATEDIF | 计算两个日期之间的时间差 |
| IF | 多层条件判断嵌套 |
| INT | 发票金额拆分 |
| VALUE | 文本型数字转数值格式 |
| MID | 身份证号自动计算生日/性别/星座/生肖/省份 |

## 技巧案例

- 利用函数生成固定格式的流水号
- 多天数预约重复性校验
- 文本与日期格式相互转换
- 日期提交校验
- 手机号码部分隐藏
- 每人每天只能提交一条数据
- 样本方差与标准差计算

## 场景实践

- BOM物料清单
- 计算去除休息时间的工作时长
- 请假天数（去除法定节假日及周末）
- 金额汇总
- 随机抽题
- 评委打分（去掉最高最低分）
- 考试自动评分系统
- 个税计算器
- 统一社会信用代码校验
- 工龄计算

## 重要提示

公式在不同功能模块（表单字段、表单校验、仪表盘计算字段）中的使用存在差异，需注意字段类型匹配。
