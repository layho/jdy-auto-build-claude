# 子表单

> 来源: https://hc.jiandaoyun.com/doc/9020

## 功能简介

子表单中可以添加多个子字段，支持录入多条数据。适用于"录入字段格式一定，但录入条数不定"的情况。

## 应用场景

- 财务管理：一个报销流程涉及多个报销款项
- 质量管理：一次抽样检查选取多个样品
- 进销存：一个订单中记录多款产品

## 子字段操作

- 支持添加、复制、删除、排序子字段
- 可添加的子字段类型：单行文本、多行文本、数字、日期时间、单选按钮组、复选框组、下拉框、下拉复选框、地址、定位、图片、附件、选择数据、查询、手写签名、成员字段、部门字段

## 子字段校验

| 校验类型 | 说明 |
|---------|------|
| 必填 | 该子字段的值必填 |
| 不允许重复值 | 所有表单数据中不允许重复 |
| 单条数据内不允许重复值 | 子表单多行数据内不允许重复 |

## 字段权限

- 可新增记录
- 可插入记录
- 可编辑已有记录
- 可删除已有记录

## 展示样式

**电脑端**：自动行高，可固定前X列
**移动端**：纵向平铺 或 横向表格

## 快速填报

开启后支持批量新增、批量编辑已有数据、复制粘贴Excel数据等。

## 注意事项

1. 一个子表单最多添加 **200 行数据**
2. 最多 200 行数据行
3. 数据存储在主表记录中
