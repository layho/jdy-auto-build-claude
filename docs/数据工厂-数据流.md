# 数据工厂 - 数据流

> 来源: https://hc.jiandaoyun.com/zh_cn/doc/10985

## 功能简介

数据工厂是加工处理数据的场所。在数据工厂中可创建多个数据流，每个数据流承担具体的数据加工和处理工作。

## 应用场景

数据流可选择多个表单作为数据源，自由组合数据处理节点，实现复杂的多表数据处理，最终得到输出表。

## 6 种数据处理节点

| 节点 | 作用 |
|------|------|
| 横向连接 | 将两张表左右相连，合并为一张新增多列的新表 |
| 追加合并 | 将多张表上下相连，合并为一张新增多行的新表 |
| 分组汇总 | 按条件将相同数据分组后进行汇总计算 |
| 数据筛选 | 添加筛选条件，过滤无用数据，只保留需要的数据 |
| 字段设置 | 调整字段顺序、控制显隐、添加新计算字段 |
| 行转列 | 将一维表转为二维表 |

## 使用条件

数据工厂属于高级功能。免费版可免费创建 1 个数据流进行体验。

## 创建入口

打开应用，点击「应用后台 >> 数据工厂 >> 新建数据流」即可创建。

## 设计界面

1. **节点选择区** — 节点可拖拽添加到数据流设计区
2. **数据流设计区** — 选中节点后，从底部弹出节点设置区
3. **节点设置区** — 对单个节点进行详细配置

## 设置步骤

数据流由「输入节点」「数据处理节点」「输出节点」构成，节点间用连接线相连。

1. **设置输入节点** — 选择要处理的数据表单及字段
2. **添加并设置数据处理节点** — 从选择区拖拽到设计区
3. **设置输出节点** — 为输出节点设置名称
4. **保存数据流** — 务必保存，并为数据流设置名称
