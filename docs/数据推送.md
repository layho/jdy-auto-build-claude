# 数据推送 (Webhook)

> 来源: https://hc.jiandaoyun.com/zh_cn/doc/11019

## 功能简介

数据推送本质上是 **Webhook（钩子）**。当表单数据/结构或流程状态/待办发生变更时，系统将变更数据推送至目标服务器。

## 设置步骤

1. 进入表单编辑页面 → 扩展功能 → 数据推送
2. 配置参数：

| 配置项 | 说明 |
|---|---|
| 目标服务器 | 自定义服务器 或 FineReport 服务器 |
| 服务器地址 | 你的服务器回调 URL |
| Secret | 用于安全验证的密钥 |
| 推送类型 | 表单变更时推送 / 流程变更时推送 |
| 推送事件 | 具体触发的事件类型 |

注意：流程变更时推送仅流程表单可用。

## 关键概念

- **Webhook**：反向 API 机制，服务器主动向你的服务器发送 HTTP 请求
- **Secret**：用于签名验签，确保推送数据来源可靠
- **推送类型**：区分表单层面变更与流程层面变更
