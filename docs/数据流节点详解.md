# 数据流节点详解

> 来源: https://hc.jiandaoyun.com/zh_cn/doc/11679

## 9 种数据流节点

| 节点类型 | 用途 |
|---------|------|
| 输入节点 | 选择要处理的数据来源 |
| 输出节点 | 数据流的出口，输出计算结果为数据流输出表 |
| 横向连接 | 数据处理 - 两张表左右相连 |
| 追加合并 | 数据处理 - 多张表上下相连 |
| 分组汇总 | 数据处理 - 分组后汇总计算 |
| 数据筛选 | 数据处理 - 过滤无用数据 |
| 字段设置 | 数据处理 - 调整字段/计算字段 |
| 行转列 | 数据处理 - 一维表转二维表 |
| 去重 | 数据处理 - 去除重复数据 |

## 节点分类

1. **输入/输出节点** — 数据的来源与出口
2. **数据处理节点** — 7种（横向连接、追加合并、分组汇总、数据筛选、字段设置、行转列、去重）

## 基本操作

- **添加**：拖拽节点即可
- **删除**：选中节点后 Delete 键
- **连接线**：节点之间必须添加连接线
- **数据预览**：查看处理后的数据
- **撤销/重做**：⌘+Z / ⌘+Y (Mac)

## 注意事项（关键限制）

1. 一个数据流最多添加 **200 个**数据处理节点
2. 横向连接、追加合并、分组汇总各最多添加 **50 个**
3. 去重节点最多添加 **10 个**
4. 已超出节点限制的数据流可正常使用，但**不可复制**
