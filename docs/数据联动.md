# 数据联动

> 来源: https://hc.jiandaoyun.com/doc/9028

## 功能简介

数据联动允许在一个表单中调用另一张表单、聚合表或数据流输出表中满足特定条件的数据。

## 应用场景

- **地址选择**：选择省份后，下一字段仅显示对应城市
- **商品信息调用**：选择商品名称/编号后自动填入对应价格

## 三个配置项

**1) 联动表单** — 选择要调用数据的另一张表单、聚合表或数据流输出表

**2) 联动条件** — 定义联动表单与当前表单字段间的匹配条件
- 多个条件关系：「所有」（且）或「任一」（或）
- 字段关系：等于、不等于、包含、不包含、为空、不为空等
- 字段值：当前表单字段 或 自定义

**3) 触发联动** — 设置当前字段要显示的内容

## 支持字段

单行文本、多行文本、数字、日期时间、下拉框、下拉复选框、图片、附件、地址、定位、子表单、成员字段、部门字段、关联数据字段

## 注意事项

1. 字段同时配置数据联动、公式编辑、AI填充、选择数据填充、前端事件回填时需排查触发逻辑
2. 下拉框/下拉复选框设置数据联动时，字段必须设为**可编辑**
3. 字段类型需匹配（单行文本↔多行文本/单选按钮组/下拉框可互通）
4. 数据联动调出多个值时需用下拉框再次选择
