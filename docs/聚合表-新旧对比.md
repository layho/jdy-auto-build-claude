# 聚合表与旧版功能对比

> 来源: https://hc.jiandaoyun.com/zh_cn/doc/22419

## 功能对照

✅=支持 ☑️=部分支持 ❌=不支持

### 数据来源

| 配置项 | 旧版 | 新版 |
|---|---|---|
| 来源表支持单独选定子表单 | ❌ | ✅ |
| 调整来源表时保留已有配置 | ❌ | ✅ |

### 维度（行表头）

| 配置项 | 旧版 | 新版 |
|---|---|---|
| 支持的字段类型 | ☑️ 单行文本、单选按钮组、下拉框、流水号、成员单选、部门单选 | ✅ 新增日期时间，可按天/周/月/季/年汇总 |
| 过滤空维度数据 | ☑️ 默认过滤，不可修改 | ✅ 支持自主配置 |

### 指标

| 配置项 | 旧版 | 新版 |
|---|---|---|
| 支持乘除及函数运算 | ❌ | ✅ |
| 多种聚合方式 | ☑️ 仅求和 | ✅ 新增平均值(AVG)及计数(COUNT) |
| 设置聚合条件 | ❌ | ✅ |

### 管理

| 配置项 | 旧版 | 新版 |
|---|---|---|
| 配置中预览全部数据 | ✅ | ☑️ 仅预览部分 |
| 不进入配置页即可预览 | ❌ | ✅ |
| 复制聚合表 | ❌ | ✅ |
| 状态可视化 | ❌ | ✅ |

## 关键差异

1. 新版支持子表单单独选定
2. 新版支持日期时间维度（按天/周/月/季/年分组）
3. 新版支持AVG/COUNT聚合方式
4. 新版支持聚合条件和函数运算
5. 新版调整来源表时保留已有配置
