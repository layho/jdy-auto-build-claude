# 聚合表

> 来源: https://hc.jiandaoyun.com/zh_cn/solution/22620

## 简介

聚合表实现跨表单数据调用功能，让用户将不同表单中的相关数据进行整合与预处理，支持更深入的数据分析和业务需求。

## 设置步骤

**1）选择数据来源**

数据来源包括多个表单，采用多表关联方式选择数据来源。

**2）将关联的字段添加为行表头并进行重命名**

当聚合表为多表关联时，行表头的选择范围是多张表单建立关联的字段。关联关系字段需要全部添加为行表头。

**3）设置指标**

指标指对表单中的字段进行计算，并将结果展示在聚合表中。

**4）添加校验条件**

点击「添加校验条件」，设置校验条件的公式。当表单数据满足条件时可提交，不满足则不允许提交并显示文字提示。

## 典型应用场景

- 实时库存统计（如备件库存，通过入库单与领用单数据汇总得出当前库存）
- 进销存/供应 > 采购管理
- WMS仓库管理
- 财务管理
- 工单管理
- 设备管理与巡检
