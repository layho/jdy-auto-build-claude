# 选择数据

> 来源: https://hc.jiandaoyun.com/doc/9024

## 功能简介

选择数据是一个高级字段，用于在当前表单中调取本表或其他表单的已有数据，并可随当前数据一同入库。可理解为功能更强的数据联动。

## 应用场景

- 人事薪资：将员工绩效、考勤信息调取到薪资表单中
- 合同管理：将客户、订单信息调取到合同明细中
- 进销存：将供应商信息同步至采购订单中

## 字段属性

### 数据源
支持：当前表单、本应用其他表单、跨应用表单、聚合表

### 数据选择过程设置
- **显示字段**：展示在数据列表中的字段（至少勾选一项）
- **数据过滤**：过滤条件限制可选数据范围
- **数据排序**：升序/降序
- **操作权限**：允许新增数据源表数据
- **快捷筛选**：最多3个筛选字段（仅PC端）

### 数据填充规则
将数据源表字段值填充到当前表单字段中，随本表数据一起提交入库。

- **填充到新字段**：系统自动创建新字段并建立对应关系
- **填充到已有字段**：手动设置对应关系

## 子表单选择数据

- 支持多选，Shift键连选（不支持跨页）
- 移动端横向表格显示时省略显示

## 注意事项

1. 选择数据字段本身不在数据管理和数据表中直接显示
2. 不支持打印，但支持打印设置的显示字段
3. 支持搜索前10个显示字段
4. 若同时配置数据联动、公式编辑、AI填充、前端事件回填，需排查触发逻辑
