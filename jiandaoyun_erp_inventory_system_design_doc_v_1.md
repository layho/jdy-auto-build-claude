# 十三、Claude/OpenClaw 技术要求

Claude 必须：

- 使用 selector registry
- 使用 smartLocate
- 使用 retry
- 使用 recovery
- 使用 validation
- 使用 networkidle fallback
- 使用 rerender recovery

---

# 十四、禁止事项

禁止：

- querySelectorAll('*')
- 自由 DOM 遍历
- 自由 Browser 探索
- React hash className
- 无限 retry
- 无限 patch loop
- page.content() 全量 DOM

---

# 十五、Browser Runtime 要求

所有 workflow：

必须：

```text
1. screenshot
2. validation
3. retry <= 2
4. recovery <= 1
5. finally close browser
```
