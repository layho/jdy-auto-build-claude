# 简道云完整知识体系

## 一、核心架构与工具

- Playwright + TypeScript (`npx tsx`) 自动化浏览器操作
- Runtime 模块: `browser.ts`（浏览器启动/会话管理）, `dom.ts`（waitForStableDOM）, `watchdog.ts`
- 会话持久化: `.temp/session.json` 避免重复登录
- API 端点: `POST /_/data_process/data/related_form/multi_create` 提交含子表数据

## 二、关键 DOM 交互模式

### 对话框选择
- 关联数据: `.fx-lookup-dialog`
- 选择数据: `.fx-linkfield-dialog`
- 点击数据行即自动关闭并确认，无需额外按钮
- `dlg.locator('tbody tr').filter({ hasText: rowText }).first()`

### 关联子表操作
- 容器: `.fx-related-form`, 列头: `.related-form-title`
- 添加行: `button.btn-add` (不在 `.fx-form.form-modal` 内)
- 快速填报: `button.btn-quick-fill`
- 输入框: `.fx-related-form input:not([type="hidden"]):not([type="checkbox"])`
- 第一列 `input-inner`, 其余 `normal-input`
- Enter 在最后输入框会新增一行（不是提交）
- 行图标: `.icon-expand`, `.icon-trash`, `.icon-copy`

### 属性面板 (编辑器)
- 面板: `.fx-design-field-config` / `.form-design-config`
- 显示字段按钮: `.config-content button:has-text("个字段")`
- 字段选择器: `.x-biz-multi-field-selector-popover`
- 复选框: `.sortable-check-item` > `input[type="checkbox"]`

### 数据查看
- 默认是录入表单页（仅添加数据模式）
- 点击顶部 "数据管理" tab (SPAN 元素) 查看数据列表
- "仅添加数据" ↔ "管理全部数据" 切换

## 三、官方文档学习 — 完整功能地图

### 3.1 表单设计 (doc/9013)
- 28种字段类型: 常用字段 + 高级字段 + 关联类型字段
- 单表单最多 500 字段
- 4步骤: 新建表单 → 添加字段 → 设置字段属性 → 设置表单属性

### 3.2 流程表单 (doc/9048, doc/12596)
- 设置步骤: 表单设计 → 流程设定 → 扩展功能 → 表单发布
- **节点负责人**: 常规审批（动态负责人/部门/成员字段）+ 逐级审批
- **节点操作**: 回退、否决、暂存、加签、转交、批量提交/审批
- **流转规则**: 节点提交条件、多位负责人策略、找不到负责人时处理、节点限时处理(自动提醒/提交/回退)、数据流转条件
- **条件流程** (doc/9061): 按条件分支流转，同时进入所有满足条件的节点
- **子流程** (doc/12149): 父流程+子流程，最多3层嵌套，支持同步/异步
- **自动提交** (doc/11196): 超时自动提交，上限30天
- **批量审批** (doc/12519): 单次最多100条
- **流程版本管理**、**流程预览/测试**

### 3.3 关联子表 (doc/21272)
- 随主数据一同新增: 仅1个关联子表可用，最多500条
- 操作权限: 可新增/可导入/可导出/可直接编辑已有数据/可删除已有数据/可查看数据详情
- 查看/编辑主数据时 与 新增主数据时 权限分离
- 快速填报: Excel复制粘贴、单元格编辑、批量编辑、下拉填充
- 数据独立存储，删除主数据不级联删除关联表数据

### 3.4 子表单 (doc/9020)
- 数据存储在主表记录中（vs 关联子表存独立表）
- 最多200行
- 权限: 可新增记录/可插入记录/可编辑已有记录/可删除已有记录
- 移动端: 纵向平铺 或 横向表格
- 支持整体数据联动 (doc/11259) 和单条数据联动 (doc/12502)

### 3.5 数据联动 (doc/9028)
- 三个元素: 联动表单 + 联动条件 + 触发联动(字段赋值映射)
- 条件支持"所有"(且)/"任一"(或) 多条件逻辑
- 字段关系: 等于、包含、大于等
- 下拉框须设为"可编辑"状态
- 与公式/AI填充/前端事件同时配置时需排查触发逻辑

### 3.6 选择数据 (doc/9024)
- 从当前表单/其他表单/聚合表/跨应用中选择已有数据
- 配置: 数据源 → 数据选择过程(显示字段/过滤/排序/快捷筛选) → 数据填充规则

### 3.7 聚合表 (doc/22408, doc/22419)
- **新版聚合表**: 单表/多表聚合，实时统计或超额提交控制
- **维度**: 最多30个，支持日期按年/季/月/周/日汇总
- **指标**: 最多100个，SUM/AVG/COUNT，支持聚合条件过滤
- **数据提交校验**: 设置公式条件（如 库存>=0）
- 支持子表单作为数据源（同一主表最多3个子表单）
- 被表单调用: 数据联动、关联数据、关联查询、仪表盘
- 新版优势: 支持AVG/COUNT(非仅SUM)、聚合条件、日期维度、子表单独立选择

### 3.8 数据工厂/数据流 (doc/10985)
- 6大处理节点: 横向连接(JOIN)、追加合并(UNION)、分组汇总(GROUP BY)、数据筛选(WHERE)、字段设置(SELECT/RENAME)、行转列(PIVOT)
- 输出表可: 同步到表单、在仪表盘中展示、在表单中调用
- 数据更新有1小时延迟
- 应用: 进销存、考勤、合同到期、工龄计算、工资计算

### 3.9 智能助手Pro (doc/18478, /solution/21997)
- **4种触发**: 表单触发、定时触发、HTTP触发、按钮触发
- **执行节点**: 新增数据(200条/次)、修改数据、删除数据、查询、条件分支(100条)、计算、HTTP请求(2025.9新增)、AI大模型、AI多模态、Agent(2026.3新增)、插件
- **限制**: 单个触发链最多新增1000条；1小时内触发树最多新增10000条
- 流程表单拆分处理（数据变更和流程变更可单独触发）
- HTTP请求节点: 零代码实现双向数据互动

### 3.10 仪表盘 (doc/13096, /solution/21995)
- 制作流程: 新建仪表盘 → 选择图表 → 选择数据源 → 统计分析 → 图表设置 → 发布
- **统计表**: 指标图、柱形图/条形图、折线图/面积图、饼图/环形图、进度图、双轴图、透视表
- **明细表**: 表格逐条展示，支持过滤、条件格式、筛选组件
- **3种数据源**: 表单数据源、数据流数据源(数据工厂)、聚合表数据源
- **流程分析表**: 7种分析事件(已创建/进行中/已完成/手动结束/平均处理时间/节点进行中/节点平均处理时间)
- **定时提醒**: 周期性推送业务数据

### 3.11 函数/公式 (doc/13956)
- **逻辑函数**: IF, AND, OR, IFS
- **文本函数**: CONCATENATE, MID, LEFT, RIGHT, REPLACE, VALUE
- **数学函数**: INT, ROUND, MAX, MIN, SUM, RANDBETWEEN
- **日期函数**: DATEDIF, DATE, YEAR, MONTH, DAY
- **高级函数**: MAPX(多条件查询), UUID, COUNT
- 应用: 身份证解析、流水号、工龄计算、随机抽题、分数计算、个税计算、校验

### 3.12 前端事件 (open/12115)
- 触发: 字段值手动修改时触发
- 支持字段: 文本/数字/日期/下拉/子表单子字段等
- 执行: 自定义请求(15s×3重试) 或 调用插件(60s不重试)
- 最多100个/表单
- 插件覆盖: 表单增强、消息通知、智能识别、信息查询、数据同步、AI、电子签章、硬件

### 3.13 跨应用 (doc/9139)
- 支持: 数据联动、公式、查询、选择数据、仪表盘、数据工厂、智能助手
- 不支持: 关联数据字段、聚合表
- 版本要求: 标准版及以上

### 3.14 数据推送/Webhook (doc/11019)
- 仅支持 POST, Content-type: application/json
- Header: `X-JDY-DeliverId`(去重), `X-JDY-Signature`(SHA-1签名)
- 需2秒内返回2xx; 重试最多5次; 连续失败100次关闭

### 3.15 权限体系 (doc/15730, doc/12407, doc/21695)
- 管理员权限: 通讯录管理 + 应用编辑
- 成员权限: 访问权限 + 数据操作权限
- 预定义权限组(普通表单): 仅添加数据、添加并管理本人数据、添加并查看全部数据、管理全部数据、查看全部数据
- 预定义权限组(流程表单): 发起流程、查看全部流程、管理全部流程
- 字段权限: 可见/可编辑

### 3.16 代码设计/插件 (open/23402)
- 支持 Python 3.10 和 Node.js 20
- 前端扩展: 调用 `$g.utils.openModal/closeModal/callFunction/openUrl`
- 后端函数 + 前端扩展组合
- 前端扩展调用后端函数上限5次

### 3.17 开放平台API (open/10993)
- 表单接口: 获取字段信息
- 数据接口: 查询/新建/修改/删除 单条或多条（批量最多100条）
- 字段ID: `_widget_` 前缀，修改字段信息不变更ID
- 日期: UTC格式或毫秒时间戳
- 图片/附件/签名URL 15天内有效
- API操作: create/update/delete 触发关联关系不同（详见对照表）

### 3.18 CRM (solution/24963, /solution/25004)
- 公海池: 自动回收规则、保有量限制、多公海
- 跟进记录: 结构化字段、AI提炼会议纪要、生成商机情报
- 客户全生命周期: 线索→公海→客户→回访→成交
- AI版: AI客户管理、AI销售周报、AI客诉工单

### 3.19 关联子表数据联动 (doc/22530)
- 字段联动: 根据主表/关联子表字段值调取其他表单数据
- 整体联动: 通过自定义按钮把表单A的关联子表数据填充到表单B的关联子表

### 3.20 打印模板 (搭建说明 solution/17156)
- 支持自定义模板个性化打印和系统模板直接打印

### 3.21 表单提交校验 (doc/12322)
- 条件公式，引用表单中字段逻辑判断
- 流程表单启用流程后校验失效，改为节点提交校验
- 节点提交条件两种模式: 所有数据可提交/满足条件才可提交

### 3.22 数据工厂详细应用案例
- BOM物料清单、多渠道订单管理、考勤表生成、工资计算、库存周转率
- 输出表在表单中调用 (doc/12386)
- 数据分析场景: 期初期末库存、累加汇总、客户RFM建模、复购率分析

## 六、各模块创建流程 (2026-05-23 实测)

### 聚合表
1. 新建聚合表 → 弹出 `fx-aggregate-source-select-dialog` 
2. 点击 `.x-biz-entry-select-combo button.add-btn` → 弹出 popover 选表单
3. 点击 `.entry-item` 选择数据源表单
4. 可选择子表单 (如订单明细表)
5. 确认后进入配置页: 数据来源 → 维度 → 指标
6. 维度: 单击"添加维度" → 多选下拉 `.x-biz-field-adder` → 选字段
7. 指标: 单击"添加指标" → `fx-aggregate-formula-edit-dialog` 公式编辑器
   - 聚合变量: 数据条数/字段 (可设聚合方式COUNT/SUM/AVG)
   - 支持逻辑函数(AND/IF/IFS/OR)和数学函数(INT/ROUND)
   - 可添加聚合条件过滤
8. 命名 + 保存
- 注意: 有子表时维度步骤变为"配置维度"(弹窗), 需配对同类型字段
- 创建了: 订单统计 (订单管理, 维度:下单日期, COUNT指标)

### 数据推送
1. 新建数据推送 → `fx-app-data-push-create-dialog`
2. 点击 `.x-biz-dropdown-label` → 选推送表单
3. 确定 → 显示完整配置:
   - 目标服务器: 自定义服务器URL
   - Secret: 可自动生成
   - 推送事件: 数据事件(新提交/修改/删除/恢复) + 表单事件(结构变化)
4. 保存后列表显示: 服务器地址、推送表单、推送事件、操作(日志/编辑/删除)
- 创建了: 推送订单管理到 https://httpbin.org/post (4个数据事件)

### 智能助手Pro
1. 新建智能助手 → `fx-automation-trigger-config-dialog`
2. 名称 + 触发方式: 表单触发/定时触发/HTTP触发/按钮触发
   - 表单触发已预选 (`.trigger-type-item.selected`)
   - 点击 `.form-selector .x-biz-dropdown-label` 选触发表单
3. 确定 → 进入工作流编辑器 (`dashboard/app/{id}/automation/{aid}/edit`)
4. 画布: 触发节点 → 动作节点 → 结束节点
5. 点击 "+" 按钮 (`.fx-automation-design-plus-icon`) → 弹出动作菜单:
   - 粘贴节点
   - 数据处理: 新增数据 / 修改数据 / 删除数据 / 计算节点
   - 更多节点
6. 节点配置抽屉: `.fx-automation-node-config-drawer`
7. 工具栏: undo/redo, zoom, 帮助文档, 联动触发, 仅保存, 保存并启用
- 创建了: 自动同步订单数据 (表单触发:订单管理, 动作待配置)

### 数据工厂
1. 新建数据流 → 全屏编辑器 (非弹窗)
2. 左侧面板: 输入输出 + 数据处理 (7种: 横向连接/追加合并/分组汇总/数据筛选/字段设置/行转列/去重)
3. 可视化DAG画布, 拖拽节点+连线
4. 引导对话框: 7步入门教学
5. 工具栏: 学习中心, 保存

### 数据推送列表操作
- 每条记录有: 日志、编辑、删除 按钮
- 支持按服务器地址、推送表单筛选

### 关键DOM交互模式 (新增)
